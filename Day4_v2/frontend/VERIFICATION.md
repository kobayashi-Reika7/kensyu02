# Day4_v2 動作確認チェックリスト

## リロード後も残らない場合の確認

1. **Firestore ルール**: [Firebase Console](https://console.firebase.google.com/) → Firestore → ルール で、`todos` コレクションの読み取りが許可されているか確認
2. **エラー表示**: 画面上に「Firestore の読み込みに失敗しました」と出る場合は、ルールまたはネットワークを確認

## 起動確認

開発サーバーは起動済みです。

- **URL**: http://localhost:5201/ （5200 が使用中の場合 5201）
- **ビルド**: ✅ `npm run build` 成功

## 手動検証手順

### 1. 初回表示

- [ ] 入力欄に「テストタスク1」と入力 → 「追加」をクリック

### 4. タブを閉じても残る
- [ ] タブを閉じる
- [ ] 再度 http://localhost:5201/ を開く
- [ ] 「テストタスク1」が**残っている**

### 5. 別タブで追加 → 即反映
- [ ] タブ A: アプリを開いたまま
- [ ] タブ B: 同じ URL を新しいタブで開く
- [ ] タブ B で「テストタスク2」を追加
- [ ] タブ A を確認 → **即座に**「テストタスク2」が表示される
