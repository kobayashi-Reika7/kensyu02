# カウントダウンタイマー 要件定義書

## 1. アプリ概要

### アプリ名
カウントダウンタイマー（test-timer）

### 目的
Webブラウザ上で、指定した時間から0秒までカウントダウンし、時間になったことを知らせるシンプルなタイマーを提供する。  
ルールのテストおよび学習用として、1つのHTMLファイルで動作する簡単な仕様とする。

### ターゲット
- 簡単な時間計測をしたい一般ユーザー
- フロントエンド（HTML/CSS/JavaScript）を学習する初心者

### 方針
- 1つのHTMLファイルで完結する
- 複雑すぎない仕様にし、初心者が読んで理解できる構造にする

---

## 2. 機能要件

### 必須機能

| 機能 | 説明 |
|------|------|
| 時間の設定 | 分（0〜99）・秒（0〜59）を入力してカウントダウンする時間を指定できること |
| 表示 | 残り時間を MM:SS 形式で画面上に表示すること |
| 開始 | 「開始」ボタンでカウントダウンを開始すること |
| 一時停止・再開 | カウントダウン中に一時停止し、再開できること |
| リセット | 「リセット」ボタンでタイマーを止め、入力欄の値に表示を戻すこと |
| 終了通知 | 0秒になったら自動で停止し、「時間です！」などで知らせること |

### 詳細仕様

- 表示は常に2桁（例：05:09）で行う
- タイマー動作中は分・秒の入力欄は無効化してよい
- 0秒以下の時間では開始できないこと（メッセージで案内してよい）
- リセット時は入力欄を再度有効にし、現在の入力値で表示を更新する

---

## 3. 非機能要件

| 項目 | 内容 |
|------|------|
| レスポンス | ボタン操作や1秒経過に応じて、即座に画面が更新されること |
| コード | JavaScriptはシンプルで読みやすく記述する（関数の目的を分ける、コメントを残す） |
| 可読性 | 初心者でも処理の流れが理解できる構造にする |

---

## 4. 画面構成（UIイメージ）

```
+------------------------------------------+
|           カウントダウンタイマー           |
+------------------------------------------+
|                                          |
|            [ 00:00 ]  （残り時間表示）    |
|                                          |
|   [ 時間です！ ]  （終了時メッセージ）    |
|                                          |
|  分 [  1 ]  秒 [  0 ]                     |
|  [ 開始 ]  [ 一時停止 ]  [ リセット ]     |
+------------------------------------------+
```

- **表示エリア**: 残り時間を MM:SS で大きく表示
- **メッセージ**: 終了時やエラー時の短文メッセージ
- **入力**: 分（0〜99）、秒（0〜59）
- **ボタン**: 開始、一時停止（再開に切り替え）、リセット

---

## 5. 扱うデータ

| データ | 説明 | 保持場所 |
|--------|------|----------|
| 残り秒数 | 現在の残り時間（秒単位で保持） | 変数（例：remainingSeconds） |
| タイマーID | setInterval の戻り値（クリア用） | 変数 |
| 一時停止フラグ | 一時停止中かどうか | 変数 |

- いずれもメモリ上のみ。ページを閉じたりリロードすると消える一時データとする
- 永続保存・履歴はスコープ外

---

## 6. 前提条件・制約

| 項目 | 内容 |
|------|------|
| 技術スタック | HTML / CSS / JavaScript のみ（1ファイルにまとめてよい） |
| 動作環境 | 一般的なWebブラウザで動作すること |
| ファイル | test-timer.html および要件定義書（requirements.md）を同一フォルダに格納する |

---

## 7. スコープ外（今回やらないこと）

- アラーム音・通知音
- 複数タイマーの同時実行
- 経過時間の保存・履歴表示
- キーボードショートカット
- 高度なアニメーションやテーマ切り替え
